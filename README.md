# Open-IP

The **easy-peasy** way for geolocation lookup for an IP Address.

💃💃💃

Example - https://open-ip.herokuapp.com/157.37.165.132

## Syntax

There is no syntax 😎.
Just add your *IP Address* as the param - https://open-ip.herokuapp.com/:ip
